## 配置参数

```
#.json直接用记事本打开#
./settings.json  
{
"order":1,
"delay":0.05
}
```
1. order=1   
按照luckyList顺序循环抽取一人  
order=0  
在luckyList中随机抽取
2. delay  
每一个元素停留时间（默认0.05s)  
**注意：delay小于0.02s会产生某些不可知的错误**
